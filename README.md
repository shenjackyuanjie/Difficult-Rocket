## 环境需求 (测试过的 / 开发平台)

- `开发平台 1 - Windows 10 x64 22H2`
  - Python `3.8.10`
    - pillow `9.5.0`
    - psutil `5.9.5`
    - rtoml `0.9.0`
    - tomlkit `0.11.7`
    - defusedxml `0.7.1`
    - objprint `0.2.2`
    - viztracer `0.15.6`
    - vizplugins `0.1.3`
    - nuitka `1.5.6`
    - ordered-set `4.1.0`
    - imageio `2.27.0`
    - wheel `0.40.0`
    - setuptools `67.6.1`
    - setuptools-rust `1.5.2`
  - `AMD R5 5600X`
  - `AMD RX 550 4G`

## 需要的 Python 模块

- `pyglet` (已经内置 V2.0.5 路径：`./libs/pyglet`)
- `xmltodict` (已经内置 V0.12.0 路径:`./libs/xmltodict`)
- `pyperclip` (已经内置 V1.8.2 路径: `./libs/pyperclip`)

```text
# this requirement is for
#   DR basic running from source
#   DR build (by nuitka)
#   DR contributing

# for images
pillow >= 9.5.0

# for sys info
psutil >= 5.9.5

# for files
rtoml >= 0.9.0
tomlkit >= 0.11.7
defusedxml >= 0.7.1

# for debug
objprint >= 0.2.2
viztracer >= 0.15.6
vizplugins >= 0.1.3

# for compile
nuitka >= 1.5.6
ordered-set >= 4.1.0
imageio >= 2.27.0
wheel >= 0.40.0
setuptools >= 67.6.1
setuptools-rust >= 1.5.2
```
