## 环境需求 (测试过的 / 开发平台)

- `开发平台 1 - Windows 10 x64 22H2`
  - `Python 3.8.10`
  - `pillow 9.3.0`
  - `pyperclip 1.8.2`
  - `pyglet 2.0`
  - `psutil 5.9.4`
  - `objprint 0.2.2`
  - `rtoml 0.9.0`
  - `xmltodict 0.13.0`
  - `tomlkit 0.11.6`
  - `AMD R5 5600X`
  - `AMD RX 550 4G`

## 需要的Python模块

- `rtoml`
- `tomlkit`
- `pyglet` (已经内置 V2.0.4 路径：`./libs/pyglet`)
- `xmltodict` (已经内置 V0.12.0 路径:`./libs/xmltodict`)
- `pyperclip` (已经内置 V1.8.2 路径: `./libs/pyperclip`)
- `pillow`
- `defusedxml`
- `objprint`
- `psutil`
